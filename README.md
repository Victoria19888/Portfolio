## Running Locally

Open `index.html` in a browser, or serve the folder with a simple static server.

Example with Python:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Notes

- The site is currently organized as a flat static project, and the asset paths in `index.html` point to files in the repository root.
- This structure works well for GitHub Pages as long as `index.html`, `style.css`, `script.js`, and the image assets stay in the same relative locations.

## Deployment

This repository is suitable for deployment with GitHub Pages from the root of the default branch.
